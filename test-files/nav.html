Test nv-001: Nav element

  • Main →
  • Products →
  • Dishwashers →
  • Second hand

Code

	<h2>You are here:</h2>
	<ul>
		<li><a href="/">Main</a> →</li>
		<li><a href="/products/">Products</a> →</li>
		<li><a href="/products/dishwashers/">Dishwashers</a> →</li>
		<li><a>Second hand</a></li>
	</ul>
</nav>
